# Student-Management-System
Description:

The Student Management System is a console-based application developed using the C programming language. The main objective of this project is to efficiently manage student records in an organized manner. It provides a simple and user-friendly interface that allows users to perform various operations related to student data.

This system enables the user to add new student details such as roll number, name, age, and course. It also allows viewing all stored student records, searching for a specific student using their roll number, updating existing student information, and deleting student records when necessary.

The application uses structures to store student data and implements functions to perform different operations like insertion, display, search, update, and deletion. The menu-driven approach makes it easy for users to interact with the system without requiring advanced technical knowledge.

Although the current system stores data temporarily using arrays, it demonstrates the fundamental concepts of programming such as data structures, control statements, and modular programming. This project can be further enhanced by integrating file handling or databases to store data permanently.

Objectives:
To manage student records efficiently
To implement basic CRUD operations (Create, Read, Update, Delete)
To understand the use of structures in C
To develop a menu-driven application
Features:
Add new student details
Display all students
Search student by roll number
Update student information
Delete student record
